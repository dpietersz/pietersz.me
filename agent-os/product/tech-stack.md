# Tech Stack

## Static Site Generator

| Component | Technology | Notes |
|-----------|------------|-------|
| Framework | Hugo | Go-based static site generator, fast builds |
| Theme | Hermit V2 | https://themes.gohugo.io/themes/hermit-v2/ - cleaner layout |
| Color Palette | Gruvbox | Custom SCSS override for Gruvbox colors |
| Content Format | Markdown | Standard markdown with Hugo shortcodes |

## Frontend

| Component | Technology | Notes |
|-----------|------------|-------|
| Typography | Monaspace Neon | GitHub Next font (https://monaspace.githubnext.com/) |
| Color Scheme | Gruvbox | Light and dark mode support |
| Syntax Highlighting | Chroma | Hugo's built-in highlighter with Gruvbox styles |
| Search | Built-in | Hermit V2 search functionality |
| Diagrams (ASCII) | GoAT | Native Hugo support for ASCII art diagrams |
| Diagrams (Complex) | Mermaid.js | JavaScript library for flowcharts, sequence diagrams |
| RSS/Atom Feed | Hugo built-in | Auto-generated at /index.xml |

## Hosting and Deployment

| Component | Technology | Notes |
|-----------|------------|-------|
| Hosting | GitHub Pages | Static site hosting from repository |
| CI/CD | GitHub Actions | Automated Hugo builds on push |
| DNS | Cloudflare | Custom domain management for pietersz.me |
| Domain | pietersz.me | Apex domain as primary, www redirects to apex |

## Content Structure

```
content/
├── posts/                    # Blog posts as page bundles
│   ├── _index.md            # Posts list page
│   └── YYYY-MM-DD-post-name/
│       ├── index.md         # Post content
│       └── images/          # Co-located images
├── about/
│   └── index.md             # About page
└── _index.md                # Homepage
```

## DNS Configuration (Cloudflare)

To point pietersz.me (apex domain) to GitHub Pages with www redirect:

### Apex Domain with A Records (Primary)
| Type | Name | Content | Proxy Status |
|------|------|---------|--------------|
| A | @ | 185.199.108.153 | DNS only |
| A | @ | 185.199.109.153 | DNS only |
| A | @ | 185.199.110.153 | DNS only |
| A | @ | 185.199.111.153 | DNS only |

### WWW Redirect to Apex
| Type | Name | Content | Proxy Status |
|------|------|---------|--------------|
| CNAME | www | pietersz.me | DNS only |

### GitHub Repository Settings
1. Go to repository Settings > Pages
2. Set custom domain to `pietersz.me` (apex domain)
3. Enable "Enforce HTTPS"
4. Ensure CNAME file with `pietersz.me` exists in repository root or is generated by build

## Build and Preview Commands

```bash
# Local development server
hugo server -D

# Production build
hugo --minify

# Build with drafts
hugo -D
```

## Performance Considerations

- Hugo generates static HTML for fast page loads
- FlexSearch runs client-side, no server dependency
- Images processed by Hugo for responsive sizes
- Cloudflare can provide CDN caching if proxy enabled (test compatibility with GitHub Pages)
- Mermaid.js loaded only when diagrams present
